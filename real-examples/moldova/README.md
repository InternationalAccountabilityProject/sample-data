Moldova
=======

OCDS output is available from the platform at http://moldova-ocds.yipl.com.np/ at paths in the format http://moldova-ocds.yipl.com.np/ocds/{CONTRACT-ID}/json

e.g. http://moldova-ocds.yipl.com.np/ocds/89270/json
